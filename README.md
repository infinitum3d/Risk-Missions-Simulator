# Risk Missions Simulator

A Python simulator for a custom **Risk** variant featuring mission-based scoring, neutral factions, and AI-controlled players.

## Project Goals

* Simulate complete games of the custom Risk variant.
* Evaluate mission balance through thousands of automated games.
* Provide deterministic simulations using random seeds.
* Generate detailed game logs and statistical reports.
* Support multiple AI strategies for future experimentation.

## Game Overview

* **3 Human-controlled AI factions**
* **3 Neutral factions**
* **42 territories**
* **12 unique mission cards**
* **Mission-based victory system**
* **Standard Risk combat and reinforcement rules with custom mission mechanics**

## Planned Features
